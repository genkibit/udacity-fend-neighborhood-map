Cafe Seeker

Description

Cafe Seeker locates coffee shops and bakeries that are within a 1000 meter radius of your location using your browser's built-in geolocation function. The goal of this project is to build a dynamic Google Maps application that demonstrates effecive use of a web service API and the MVVM JavaScript framework knockout.

Features

  • Responsive full-screen Google Map
  • Automatic filtering of coffe shops
  • Alphabetical listing of places for easy manual search
  • Detailed information sourced from Google Place Search API
  • Memo functionality
  • Error indicator
  • Clean and responsive user interface
